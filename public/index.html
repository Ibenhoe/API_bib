API Documentatie

Welkom bij de documentatie van de API.

Endpoints

Boeken

  • GET /books - Haal alle boeken op.
    Voorbeeldresponse: 
    [
    {
        "_id": "63b6f7e593d87e72f4ecb3d4",
        "title": "De Hobbit",
        "author": "J.R.R. Tolkien",
        "publishedYear": 1937,
        "genre": "Fantasy",
        "available": true
    }
]
  • POST /books - Voeg een nieuw boek toe.
    Body (JSON): 
    {
    "title": "De Hobbit",
    "author": "J.R.R. Tolkien",
    "publishedYear": 1937,
    "genre": "Fantasy",
    "available": true
}
  • PUT /books/:id - Werk een bestaand boek bij.
    Body (JSON): 
    {
    "title": "De Hobbit (Updated)"
}
  • DELETE /books/:id - Verwijder een boek.

Auteurs

  • GET /authors - Haal alle auteurs op.
  • POST /authors - Voeg een nieuwe auteur toe.
    Body (JSON): 
    {
    "name": "J.R.R. Tolkien",
    "bio": "Schrijver van De Hobbit en De Lord of the Rings",
    "birthDate": "1892-01-03"
}

Validatie

Let op: De volgende regels gelden voor het invoeren van gegevens:

  • Titel en auteur zijn verplicht voor boeken.
  • Het publicatiejaar moet een geldig getal zijn.
  • Naam van de auteur is verplicht.
  • Geboortedatum moet een geldige datum zijn.
