API Documentatie

Deze API biedt functionaliteiten om boeken en auteurs te beheren. Hieronder staan alle beschikbare endpoints beschreven.

Boek Endpoints

1. Alle boeken ophalen

URL: GET /books

Query Parameters:

  • limit (optioneel) - Maximaal aantal resultaten (standaard: 10).
  • offset (optioneel) - Startpunt in de resultaten (standaard: 0).
  • sortBy (optioneel) - Veld waarop gesorteerd wordt (standaard: title).
  • order (optioneel) - Sorteervolgorde (asc of desc).

Voorbeeld:

GET /books?limit=5&offset=0&sortBy=publishedYear&order=desc

2. Boeken zoeken

URL: GET /books/search

Query Parameters:

  • title (optioneel) - Zoek op titel.
  • author (optioneel) - Zoek op auteur.

Voorbeeld:

GET /books/search?title=Harry Potter&author=J.K. Rowling

3. Nieuw boek toevoegen

URL: POST /books

Body Parameters (JSON):

  • title (verplicht) - Titel van het boek.
  • author (verplicht) - Auteur van het boek.
  • publishedYear (verplicht) - Jaar waarin het boek is gepubliceerd.

Voorbeeld:

POST /books
{
    "title": "The Great Gatsby",
    "author": "F. Scott Fitzgerald",
    "publishedYear": 1925
}

Auteur Endpoints

1. Alle auteurs ophalen

URL: GET /authors

Voorbeeld:

GET /authors

2. Nieuwe auteur toevoegen

URL: POST /authors

Body Parameters (JSON):

  • name (verplicht) - Naam van de auteur.
  • birthDate (verplicht) - Geboortedatum in ISO8601-formaat (YYYY-MM-DD).

Voorbeeld:

POST /authors
{
    "name": "George Orwell",
    "birthDate": "1903-06-25"
}
